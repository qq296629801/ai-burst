# 组织经验与模块复用（节选）

> 完整版：[多Agent协作与项目管理产品线需求.md](../../多Agent协作与项目管理产品线需求.md)

## §2.1 模块说明

| 模块 | 说明 |
|------|------|
| 组织经验与模块复用 | 已归档发布版本进入**组织级可检索知识**；新项目可按功能模块**匹配历史共性部分**并**引用/复制蓝图**；与**项目隔离**策略配套 |

---

## §1 产品目标（相关条）

- 项目**交付并发版上线**后，支持**归档发布版本**，沉淀经验供后续新项目参考；**项目之间数据与会话严格隔离**；新项目可**按需引用**已归档版本中的**经验与可复用模块信息**（只读或受控复制）。

---

## §5.8 项目隔离与跨项目经验共享

**隔离（硬约束）**

- **项目维度强隔离**：不同项目的任务、需求草稿、配置、**Agent 会话与上下文存储**、运行中协作线程**不得跨项目混用或默认互通**。
- **新项目的 Agent 会话**一律归属**新项目空间**。

**共享（受控、只读或可复制的知识层）**

- 组织知识库来源与回流见 [组织知识库](../组织知识库/需求.md)。
- 新建项目时可授权**检索与订阅**组织知识库及归档提炼物；功能模块匹配时支持**一键引用**（只读）或**复制为本项目模块蓝图/任务模板**（复制后属于新项目资产）。
- **新项目 Agent** 可使用被授权的组织经验作为**外部知识注入**，但**不得**将其他项目的**私有会话原文**默认注入。

---

## §9 验收要点（与本模块相关）

- **不同项目会话与数据互不影响**；新项目可**引用/复制**历史归档中的**共性模块蓝图**，且新项目 Agent 在**本会话空间**内可使用**被授权的组织经验**辅助工作。
