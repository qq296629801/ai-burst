# 协作与沟通可观测（节选）

> 完整版：[多Agent协作与项目管理产品线需求.md](../../多Agent协作与项目管理产品线需求.md)

## §2.1 模块说明

| 模块 | 说明 |
|------|------|
| 协作与沟通可观测 | Agent 间消息/协作线程对用户可见（可按权限脱敏）；支持按项目、按 Agent、按时间筛选 |

---

## §4.4 沟通可观测

- Agent 间协作消息（请求派工、**阻塞说明**、澄清需求、联调约定等）需进入**项目沟通时间线**或**按线程分组**的视图。
- 支持人工介入：用户可 @ 某 Agent 或插入备注，参与决策（权限可控）。

---

## §1 产品目标（相关条）

- 提供**项目级工作台**：从项目进入可查看参与 Agent 列表、**沟通记录**与任务状态。

---

## §9 验收要点（与本模块相关）

- 可观察到 **Agent 间沟通记录**（模拟或真实编排均可，以产品演示路径为准）。
