# 改进记录（节选）

> 完整版：[多Agent协作与项目管理产品线需求.md](../../多Agent协作与项目管理产品线需求.md)

## §2.1 模块说明

| 模块 | 说明 |
|------|------|
| 改进记录 | 每个 Agent 的变更、调优、反馈与版本记录可追溯 |

---

## §1 产品目标（相关条）

- **全量记录**各 Agent 的行为与**改进轨迹**（含版本或迭代说明）。

---

## §5.3 改进记录（必录）

- 对每个 Agent：**配置变更**、**提示词/策略版本**、**人工反馈采纳**、**输出质量标注**等均写入**改进日志**。
- 日志需可检索、可按 Agent / 项目 / 时间过滤；支持导出（可选）。

---

## §9 验收要点（与本模块相关）

- 每个 Agent 的**改进记录**可查询。
