## §1 产品目标（相关条）

- 提供**运营大屏**：实时动态展示 Agent **是否在干活**、**产出效率**与异常；对**停摆**、**升级未解**等场景**告警**。

---

## §5.5 运营大屏（实时动态）

- 提供**大屏视图**（支持常规大屏分辨率与全屏展示；可与普通项目看板共用数据源）。
- **实时动态**展示（刷新策略由技术方案约定，产品侧要求「观感实时」）：
  - 各 Agent **当前状态**：空闲 / 执行中 / 等待输入 / 阻塞 / 离线或异常；
  - **是否在干活**：与当前绑定任务、最近产出时间戳、心跳等关联展示；
  - **产出效率**：至少展示项目可理解的指标（如周期内完成任务数、交付物更新频次、可选 Token/调用成本等），支持按 Agent、按职能线、按项目筛选。
- **告警**：**Agent 应干活但未干活**；**项目经理 Agent 无法协调**时**大屏告警**并保留事件编号。

与升级链路联动见 [阻塞与升级链路](../阻塞与升级链路/需求.md) §5.6。

---

## §9 验收要点（与本模块相关）

- **运营大屏**可实时或近实时展示 Agent **是否在干活**与**产出效率**；停摆与项目经理**无法协调**时出现**可见告警**。
